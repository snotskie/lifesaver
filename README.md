# lifesaver
tired of losing my 2fa key because my phone broke

[view it with this rawgit link](https://rawgit.com/snotskie/lifesaver/master/).

enter your secret key (base 32 or base 16).

configure a pattern.

don't forget it.

ever.

the website will give you a url. save that url to your favorites.

visit that url and input your pattern to unlock the authenticator. if you input the wrong pattern,
it'll still try to work, but the codes will be all wrong.
